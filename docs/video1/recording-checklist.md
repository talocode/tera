# Video 1: Recording Checklist

## Before You Record

- [ ] Close all browser tabs except the ones below
- [ ] Close Slack, Discord, email, phone notifications
- [ ] Set browser zoom to 110% (Cmd/Ctrl + +)
- [ ] Clear Tera chat history (start fresh)
- [ ] Log in to Tera with Google account
- [ ] Test screen recording (1920x1080, 30fps)
- [ ] Test microphone (if doing voiceover)
- [ ] Have prompts.txt open in separate window for copy-paste

## Exact Browser Tabs to Open

| Tab | URL | Purpose |
|-----|-----|---------|
| 1 | `teraai.chat` | Homepage — show the dark UI |
| 2 | `teraai.chat` | Fresh chat — for prompt 1 (understand) |
| 3 | `teraai.chat` | Same chat — for prompts 2-4 (quiz) |
| 4 | `teraai.chat` | Same chat — for prompts 5-6 (study plan) |
| 5 | `teraai.chat` | Same chat — for prompts 7-8 (summarize) |

**Note:** All prompts go in the same chat session. Tera remembers context — this is a feature to show.

## Browser Settings

- **Font size:** 110% zoom
- **Bookmarks bar:** Hidden (Cmd/Ctrl+Shift+B)
- **Extensions:** Disable all visible ones
- **Notifications:** Do Not Disturb on
- **Window:** Full screen, no tabs visible
- **Profile:** Use a clean Chrome profile (no personal bookmarks/history)

## Recording Software Settings

| Setting | Value |
|---------|-------|
| Resolution | 1920x1080 |
| Frame rate | 30fps |
| Format | MP4 (H.264) |
| Audio | Microphone + system audio (if doing voiceover) |
| Pointer | Highlight clicks (yellow circle) |
| Area | Full screen or browser window only |

## During Recording

- Type slowly — viewers need to read the prompts
- Pause after Tera responds — let the output sink in
- Use cursor to highlight — draw attention to key parts
- Keep browser clean — no bookmarks bar, no extensions visible
- Record in one take if possible — cuts are fine but flow matters
- Do 2-3 takes of the hook and CTA

## After Recording

- [ ] Stop screen recording
- [ ] Save raw file as `tera-video1-study-with-ai-raw.mp4`
- [ ] Review footage for issues
- [ ] Note timestamps for editing cuts
- [ ] Record voiceover sections separately (if doing voiceover)
- [ ] Save voiceover as `tera-video1-voiceover.wav`
