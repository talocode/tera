## Text Style Guide

- Font: Inter (or similar sans-serif)
- Main text: #FFFFFF (white)
- Accent text: #00FF88 (neon green)
- Background: #0A0A0A (near black) for step titles
- Fade in: 0.3s
- Fade out: 0.3s
- Position: centered for step titles, bottom-left for callouts
