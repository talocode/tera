## Image Direction

**Option A: Screen recording crop**
- Crop the Tera chat interface (dark background, green accent)
- Place in bottom-right corner
- Shows the product without being cluttered

**Option B: Student silhouette**
- Dark silhouette of student at desk
- Green glow from laptop screen
- Abstract, clean, no face needed

**Option C: Text only**
- No image, just bold text on dark background
- "STUDY FASTER" dominates
- Clean, minimal, high contrast

## Creation Steps

1. Create new design: 1280x720px
2. Set background to #0A0A0A
3. Add text "STUDY" — Inter Bold, 120pt, #FFFFFF, top-left
4. Add text "FASTER" — Inter Bold, 120pt, #FFFFFF, below "STUDY"
5. Add text "with AI" — Inter Regular, 60pt, #00FF88, below "FASTER"
6. Add Tera logo — top-right or center-right
7. Add subtle green glow behind text (low opacity circle)
8. Add student silhouette or screen crop in bottom-right
9. Export as PNG, 1280x720, under 2MB
