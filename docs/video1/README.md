## Quick Start

### If you have voice:

1. Open `recording-checklist.md` — set up browser
2. Open `voiceover-script.md` — record 8 sections
3. Open `recording-plan.md` — record screen while reading script
4. Open `thumbnail-brief.md` — create the thumbnail
5. Open `publishing-checklist.md` — upload to YouTube

### If you don't have voice:

1. Open `recording-checklist.md` — set up browser
2. Open `no-voiceover-version.md` — follow the visual-only script
3. Record screen with text overlays only
4. Add ambient background music
5. Open `thumbnail-brief.md` — create the thumbnail
6. Open `publishing-checklist.md` — upload to YouTube

## Prompts to Type During Demo

```
Explain photosynthesis like I'm 10 years old. Use a simple analogy a kid would understand, then give me 3 real-world examples.
```

```
Now give me 3 real-world examples of photosynthesis I can see in everyday life.
```

```
Create 5 quiz questions about photosynthesis. Mix multiple choice and short answer. Include the answers at the end.
```

```
I got questions 2 and 4 wrong. Explain why those answers are correct and give me a memory trick to remember them.
```

```
I have a biology exam in 7 days. Create a 7-day study plan that covers cell biology, genetics, and evolution. Include 1 hour per day, with a mix of reading, practice questions, and review. Make it realistic — I have school during the day.
```

```
Can you make day 3 more focused on genetics? That's my weakest area.
```

```
I have notes on photosynthesis. Summarize them into 5 key points I need to remember for the exam. Format as bullet points.
```

```
Turn these 5 points into flashcards with the concept on one side and the explanation on the other.
```
